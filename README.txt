Данная программа создает комментарии в случае нахождения клучевых слов.
Ключевые слова:
"cout <<"
"cin >>"
"if ("
"else if ("
"else"
"for"
"while"
"do"
"return"
"char/int/float/double/string"
#include

Обратите внимание, что ключевые слова обязаны быть отделены от других слов и символов пробелом!
т.е. пробел между cout/cin/if и т.п. и "<<" / ">>" / "(" обязательно должен сохранятся.

Чтобы создать комментарии:
- Поместите файл с кодом в директорию, где находится данное приложение.
- Переименуйте ваш файл в "Code.cpp" (обязательно).
- Запустите приложение.
- Ваш код с комментариями будет сохранен в файле из этой же директории (он будет создан в ходе работы программы) под названием "Code_Com.cpp"
